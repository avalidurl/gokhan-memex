# NUMETAL

NUMETAL is an agent accelerator studio under [Atelier Gökhan](https://gokhan.vc). It builds and backs autonomous agents across four markets: onchain, compute, artistic & cultural, and frontier-lab model toolkits.

Chain-agnostic and home on Base — where the token, treasury, and settlement live — with agents reaching demand wherever it is and bringing value back.
