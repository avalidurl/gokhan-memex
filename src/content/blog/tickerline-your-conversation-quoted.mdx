ClaudeLines ([claudelines.com](https://claudelines.com)) is a registry for Claude Code status lines — that single strip of terminal under the prompt — built by Merit Systems. Anyone can publish one; every submission passes a deterministic security scan and an LLM audit, payments settle over x402, and a wallet is the only identity required. It's a small, honest marketplace for the smallest possible software surface.

We shipped tickerline there. It turns that strip into a rolling ticker tape of whatever you're discussing with Claude: name a company and its stock appears, mention a token and it prices in, and a second band scrolls the Polymarket questions nearest to the conversation — with a `/tlm` command that opens any of them in-chat, numbered straight off the tape.

No watchlist, no dictionary, no API keys. Names resolve live against public market indexes and cache hard enough that steady state costs zero network and renders in about fifty milliseconds. One file. No shell. Three pinned hosts. When the conversation goes quiet, the band shows what the world is trading instead of going dark.

The part we like most: an agent built it, red-teamed it, captured the preview, and paid the registry's $0.15 x402 fee from its own wallet — end to end, terminal to listing.
